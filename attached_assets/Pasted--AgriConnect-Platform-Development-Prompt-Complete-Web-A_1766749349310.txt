🌾 AgriConnect Platform Development Prompt - Complete Web Application
Executive Brief
You are tasked with building a revolutionary $1M+ agricultural marketplace platform that connects farmers directly with buyers, eliminates intermediaries, and creates dignified economic roles across all age groups (0-70+). This is a comprehensive, full-stack web application combining Amazon-level e-commerce with real-time mapping, AI automation, multi-currency support, and complete agricultural taxonomy coverage.
 
🎯 Core Platform Vision
Build a modern, scalable agricultural e-commerce ecosystem featuring:
•	✅ Zero-education farmer accessibility (icon-based, voice-enabled, photo-upload selling)
•	✅ Real-time satellite map integration showing nearby farmers and products
•	✅ Complete agricultural taxonomy (crops, livestock, inputs, services, daily essentials)
•	✅ Multi-region, multi-currency support for worldwide access
•	✅ AI-powered product suggestions, demand forecasting, and inventory management
•	✅ Integrated logistics, payment escrow, and trust/reputation systems
•	✅ Government scheme integration and financial inclusion tools
